# 快递柜取件码管理系统

一个简单易用的快递柜取件码管理工具，帮助您轻松管理多个快递柜的取件信息，实时监控包裹状态，避免超时费用。

## 功能特性

- 📦 **批量导入** - 支持粘贴快递信息，自动解析取件码、快递柜名称、存放时间等
- ⏰ **实时监控** - 自动计算剩余免费时长，提醒即将超时的包裹
- 🏷️ **智能分类** - 按快递柜自动分组显示，一目了然
- ✅ **已取标记** - 标记已取包裹，记录取件时间，避免重复查看
- ⚙️ **自定义设置** - 支持添加不同快递柜及其免费存放时长
- 💾 **本地存储** - 数据保存在浏览器本地，刷新不丢失
- 📱 **响应式设计** - 完美适配手机和电脑屏幕

## 使用方法

### 1. 导入快递信息

在"数据导入"文本框中粘贴快递信息，支持以下格式：

```
12月25日 14:30
【丰巢】取件码123456
免费存放18小时
```

或简化格式：

```
14:30
【蜜罐】取件码789012
```

点击"导入数据"按钮即可自动解析。

### 2. 查看包裹状态

包裹按快递柜分组显示，状态标识：
- ✅ **正常** - 绿色，剩余时间充足
- ⏰ **即将超时** - 橙色，剩余时间少于2小时
- ⚠️ **已超时** - 红色，已超过免费时长

### 3. 标记已取

点击包裹卡片右侧的"标记已取"按钮，包裹将显示为绿色半透明状态，并记录取件时间。

### 4. 自定义快递柜设置

点击右上角"⚙️ 设置"按钮，可以：
- 添加新的快递柜名称和免费存放时长
- 删除不需要的快递柜设置

预设快递柜：
- 丰巢：18小时免费
- 蜜罐：72小时免费
- 和驿智能柜：24小时免费

### 5. 清空数据

点击"清空数据"按钮可以清除所有包裹信息（需确认）。

## 开发说明

### 本地运行

直接用浏览器打开 `index.html` 文件即可使用，无需服务器环境。

### 数据格式

包裹数据结构：
```javascript
{
  id: 1234567890,           // 唯一ID
  date: Date,              // 存放时间
  lockerName: "丰巢",       // 快递柜名称
  code: "123456",          // 取件码
  freeHours: 18,           // 免费时长
  picked: false,           // 是否已取
  pickedTime: null,        // 取件时间
  originalText: "..."      // 原始文本
}
```
